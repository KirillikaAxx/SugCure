# SugCure++

SugCure++ is a reimplementation of the classic Scratch-based SugCure game, rewritten in C++ using SFML 2.6.

---
## Screenshot
![Logotype](logotype 2.png)

---
## Features
- C++ implementation
- SFML-based rendering
- Platformer physics (AABB collisions)
- ImGui debug / level editor
- Tile-based background system
---
## Controls
- A / D — move left / right
- W — jump
---
### Requirements:
- SFML 2.6+
- C++17 or newer
